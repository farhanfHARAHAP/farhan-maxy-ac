## Installation
1. Run Apache and MySQL in XAMPP
2. Import this project files to htdocs
3. Open PhpMyAdmin
4. First, create table called "h2"
5. Second, run MySQL code in /database/migration.sql
6. Third, run MySQL code in /database/seeder.sql
7. Done!
